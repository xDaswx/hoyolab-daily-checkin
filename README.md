# Hoyolab Daily Check In


# Required cookie :
- _MHYUUID
- cookie_token
- account_Id

# How to get cookie :
- Go to Hoyolab Official website 
- Right Click on website page > Inspect element > Go to network tab ..
- Login using your Genshin account
- Search cookie
- Save required cookie
